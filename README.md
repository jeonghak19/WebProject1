# Hello Spring

Spring 을 사용한 게시판 토이프로젝트

## 사용기술
- Spring Boot
- JdbcTemplate
- Spring Data JPA
- Spring Validation
- Lombok
- Thymeleaf
- MySql
- Bootstrap
 
## 기능
- 회원가입, 로그인
- 게시판 등록, 수정, 삭제(관리자)
- 게시물 등록, 수정, 삭제, 조회(회원)
- 마이페이지
- 게시물 검색
- 조회수, 좋아요 기능
- 페이지 처리
- 파일 추가

## 화면 레이아웃
- frontend
> cd frontend && npm run start

- backend
> mvn spring-boot:run
